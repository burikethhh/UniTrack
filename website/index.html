Powerful Tracking Features

Comprehensive tools for real-time faculty location monitoring and accountability

Live GPS Tracking

Monitor faculty locations in real-time with high-precision GPS. View exact positions on the interactive campus map with continuous location updates.

Faculty Monitoring Dashboard

Administrators can view all faculty members' current locations, track movement history, and monitor presence within assigned campus buildings and offices.

Smart Notifications

Receive instant alerts when faculty enter or leave designated zones, miss scheduled office hours, or when tracking sessions begin and end.

Comprehensive Reports

Generate detailed location history reports, office hour compliance summaries, and movement analytics with export options for PDF and Excel.

Secure Access Control

Installation Guide

Follow these simple steps to install UniTrack on your device

1

Download the APK

Click the download button above to get the latest version of UniTrack APK file to your device.

2

Enable Unknown Sources

Go to Settings → Security → Install unknown apps and enable installation from your browser or file manager.

3

Install the App

Open the downloaded APK file and tap "Install". Wait for the installation to complete.

4

Grant Permissions

Open UniTrack and grant the required permissions for location (important for tracking), notifications, and storage when prompted.

5

Login & Start Tracking

Use your SKSU email credentials to login. Faculty members will have their location tracked, and department heads can monitor faculty positions on the live campus map.

System Requirements

  • Android 6.0 (Marshmallow) or higher
  • At least 150MB free storage
  • GPS/Location services enabled (required for tracking)
  • Internet connection (WiFi or mobile data)
  • Valid SKSU faculty/staff email account
